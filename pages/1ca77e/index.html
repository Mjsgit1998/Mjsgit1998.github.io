# 滚动栏样式

		&::-webkit-scrollbar {
          /*滚动条整体样式*/
          width : 5px;  /*高宽分别对应横竖滚动条的尺寸*/
          height: 1px;
        }
        &::-webkit-scrollbar-thumb {
          /*滚动条里面小方块*/
          border-radius: 10px;
          box-shadow   : inset 0 0 5px #d4e2fe;
          background   : #d4e2fe;
        }
        &::-webkit-scrollbar-track {
          /*滚动条里面轨道*/
          box-shadow   : inset 0 0 5px #f2f5fd;
          border-radius: 10px;
          background   : #f2f5fd;
        }
